# Source Trust Ratings

## Tier 1 (High Trust)

- Official company websites
- SEC filings (EDGAR)
- Crunchbase (with verification)
- LinkedIn (public profiles)
- GitHub (verified repos)
- Government business registries

## Tier 2 (Medium Trust)

- Tech news (TechCrunch, The Information)
- Industry publications
- Press releases (cross-reference)
- Podcast interviews
- Verified newsletters

## Tier 3 (Verify Before Using)

- Social media claims
- Forum discussions
- Blog posts
- Anonymous sources
- User reviews (unverified)

## Confidence Ratings

| Rating     | Criteria                                   |
| ---------- | ------------------------------------------ |
| **High**   | Multiple Tier 1 sources confirm            |
| **Medium** | One Tier 1 + supporting Tier 2             |
| **Low**    | Only Tier 2/3 sources, or conflicting info |
| **Verify** | Single source, needs confirmation          |

## Source Verification Checklist

- [ ] Check domain age and reputation
- [ ] Verify author credentials
- [ ] Cross-reference with Tier 1 sources
- [ ] Check publication date
- [ ] Look for original vs. syndicated content
