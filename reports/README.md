# Reports Directory

**Purpose**: Store permanent analysis reports, findings, and documented learnings.

**Contents**:
- Analysis reports from agent investigations
- Post-mortems and incident reports
- Performance and metrics reports
- Research findings

**Format**: Markdown files with consistent naming: `YYYY-MM-DD-description.md`

**Rules**:
- Files here ARE committed to git (permanent record)
- Use for findings that should persist across sessions
- Link to related lessons in LESSONS.md
