## Blocker Types

- dependency: Waiting on another agent's deliverable
- error: Critical error blocking progress
- question: Needs clarification/decision
- resource: Missing resource or access

## Escalation Path

1. Document in this file
2. Notify in coordination channel
3. If blocked >2 hours: Escalate to project lead
